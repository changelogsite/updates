You can change the categories as per your need.
Default categories are:

| Category      | Color         |
| ------------- |:-------------:|
| New           | blue          |
| Fix           | orange        |
| Update        | green         |
| Improvement   | purple        |


To change the category you can update `categories.yml` file under `_data` folder

``` yml
# Available colors: blue, red, orange, green, purple

- name: New
  color: blue

- name: Fix
  color: orange

- name: Update
  color: green

- name: Improvement
  color: purple
```
